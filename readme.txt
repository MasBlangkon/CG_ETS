# Project Title
Animasi 2D: Japanese Crest & Scandinavian Pattern.

#Description
Program ini menampilkan Japanese Police Crest dan Semicircled Retro Scandinavian Styles. 
Dengan menerapkan penggunaan transformasi rotasi dan dilatasi (scaling) pada Japanese Police Crest.

#Author
Febriyoga Bagus Satria
Mahasiswa D3 Teknik Informatika
Politeknik Negeri Bandung

### Prerequisites
1. Telah meng-install IDE yang dapat meng-compile program C, seperti Dev C++ atau CodeBlocks.
2. Telah memasang library winbgi dan graphics.h

### Installing
1. Buka file project: "ETS_181511013.dev"
2. Pastikan pengaturan project telah mengikuti pengaturan untuk meng-compile winbgi
    seperti di laman berikut https://www.cs.colorado.edu/~main/bgi/dev-c++/
3. Compile.

###Test
1. Saat program berjalan pengguna cukup meng-input nilai lebar layar dan tinggi layar sebuah window graphic yang akan ditampilkan.
